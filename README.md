# mkgateway-lab

一个用于归档与发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放各类独立开发的 HTML 页面，每个页面通常对应一个独立的功能或展示场景。页面之间无相互依赖关系，各自可独立运行。适用于轻量级工具页面、原型演示、静态内容展示等场景。

仓库本身不绑定任何特定域名或网站，仅作为页面文件的存储与版本管理用途。

## 目录结构

```
.
├── README.md          # 本文件
└── pages/            # 存放所有 HTML 页面文件
    ├── page1.html    # 示例页面（实际名称以内容为准）
    └── ...
```

页面文件统一存放于 `pages/` 目录下，文件名应反映页面功能或用途。

## 页面归档说明

- 每个 HTML 文件均为独立的、可自包含的页面，无外部资源依赖（除非在页面内显式引用）。
- 页面内容由提交者自行维护，本仓库不保证所有页面均持续可用或保持更新。
- 页面文件命名建议简洁明了，避免使用特殊字符或空格。
- 若页面需要额外资源（如图片、样式表、脚本），建议内联或放置于同一目录下。

## 维护说明

- 欢迎通过 Pull Request 提交新的独立 HTML 页面，或对现有页面进行改进。
- 提交前请确保页面可在主流浏览器中正常运行。
- 请勿在页面中嵌入恶意代码、跟踪脚本或违反法律法规的内容。
- 本仓库为个人项目归档用途，不提供技术支持或 SLA 承诺。
